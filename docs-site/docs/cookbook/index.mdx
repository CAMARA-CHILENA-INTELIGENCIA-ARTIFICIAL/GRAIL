## Estructura típica de una receta

Cada receta sigue el mismo formato:

1. **Caso de uso** — qué problema resuelve.
2. **Stack** — qué piezas usas (modo, vector store, LLM, etc.).
3. **Paso a paso** — código completo, comandos, configuración.
4. **Pruébalo** — un comando o sample con el que verificar que funcionó.
5. **Extender** — cómo lo adaptarías a un caso similar.
