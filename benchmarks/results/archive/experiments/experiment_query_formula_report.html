The Query Formula

How the agent should craft local_search queries for maximum entity matching

WHO + WHAT + SPECIFIC TERMS

[WHO does it] + [WHAT is the process/concept] + [SPECIFIC TERMS from entity descriptions]

Local search embeds the query and matches against entity descriptions (name: description retrieval_queries). More term overlap = higher cosine similarity = better entity retrieval.

BAD: "comité o comisión que decide qué enfermedades están cubiertas" → 1/5 keywords

GOOD: "proceso del Ministerio de Salud para elaborar la propuesta de garantías explícitas en salud, basado en estudios epidemiológicos, carga de enfermedad, evaluaciones económicas y costo-efectividad" → 5/5 keywords
